# FE-05: Accessible Component Fundamentals

This assignment implements accessible interactive components from scratch following WAI-ARIA Authoring Practices, along with a comparison against the `shadcn/ui` library.

## Location
All assignment code is located in the `playground/` directory, which is a standalone React + TypeScript Vite app.

## Components Built
1. **Handmade Modal Dialog**: Features focus trapping, focus return on close, Escape key support, and correct `role="dialog"` attributes.
2. **Handmade Tabs**: Features roving `tabIndex`, arrow key navigation, Home/End key support, and proper `tablist`/`tab`/`tabpanel` attributes.
3. **Handmade Disclosure**: Uses native button behavior for expand/collapse, toggling `aria-expanded` and `aria-controls`.
4. **shadcn/ui Comparison**: The app includes a section comparing the handmade components with `shadcn/ui`'s Dialog and Tabs components.

## How to Run the Playground

```bash
cd playground
npm install
npm run dev
```

## How to Build and Verify

```bash
cd playground
npm run build
npx tsc --noEmit
```
